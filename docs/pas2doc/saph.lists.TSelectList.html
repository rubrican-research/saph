GItemList = specialize TFPGObjectList<GItem>;

To return lists from this class. To use, you have to declare variables as: <SpecializedClassName>.GItemList

function listAll: GItemList;

Remove an item from the list. Requires the item object as a parameter Returns a specialized TFPGObjectList object with the items in the same order as the main list. The list object does not own the items. You have to free the list object after you are done.
