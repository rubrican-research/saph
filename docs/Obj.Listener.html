Types

TListenerProc = procedure(const _sender: TObject; const _event: string; constref _params: TJSONObject);

These are Event Listeners. Implement procedures with this signaturs in the main program and then assign them to the event using addListener. In the implementation of the procedure, you have access to: _sender: This is the Control that sent the signal - the control on which addListener was called _event: The event. Text. You can implement a case structure to handle multiple events. _params: Parameters as a JSONObject. DO NOT free the object inside your listener procedure!! The runner with free it after the procedure is called.
